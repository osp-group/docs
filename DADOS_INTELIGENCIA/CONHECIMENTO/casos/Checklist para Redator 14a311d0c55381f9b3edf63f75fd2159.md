Para começar a escrever um artigo para blog otimizado para SEO, siga o checklist abaixo.

# Passo a passo

## 1. Entenda o tema

[Como escrever conteúdo de qualidade](Como%20escrever%20conte%C3%BAdo%20de%20qualidade%2014a311d0c55381d581a2f4b685ed7fee.md)

- [ ]  Qual é o tema do texto enviado pelo planejamento?
- [ ]  Como aplicar uma linguagem impessoal, simples e assertiva?
- [ ]  Como atingir o maior número de pessoas com essa informação?
- [ ]  Como explicar esse conteúdo da forma mais didática e inclusiva?
- [ ]  Qual a principal mensagem do texto? O que é mais importante?

## 2. Siga os princípios

- [ ]  Conteúdo original
- [ ]  Conteúdo organizado
- [ ]  Conteúdo simples
- [ ]  Conteúdo dinâmico
- [ ]  Conteúdo rápido

## 3. Pesquise no Google

- [ ]  O que os concorrentes do Google fizeram? Quais pontos fortes e fracos?
- [ ]  Quais são as referências, dados e fundamentos para sustentar o tema?
- [ ]  O que é preciso pesquisar para preencher os tópicos do planejamento SEO?
- [ ]  Como fazer o melhor conteúdo publicado na internet sobre o tema?
- [ ]  É possível adicionar novos ângulos ao tema, trazendo listas, passo a passo, infográficos e mapas mentais para enriquecer o conteúdo?
- [ ]  Nos EUA ([Google.com](https://www.google.com/?gl=us&hl=en&pws=0&gws_rd=cr)), como esse tema é abordado?
- [ ]  Como fazer um conteúdo realmente original?

## 4. Use as palavras-chaves

- [ ]  Use as palavras-chaves em harmonia com a semântica do texto
- [ ]  Organize as palavras-chaves relacionadas no corpo do texto
- [ ]  Faça a formatação com as devidas configurações de *headings* (veja exemplo abaixo)

[](https://lh7-us.googleusercontent.com/docsz/AD_4nXdvPsbkPhveKxsmU9KvpkhZh_W4eqVUseA0FO28M7lhWOlu1Zilcp-q5ebVjpb4uFp4twVkvKmTr2J0MKbToTfsgYq0NikDGiq7Ht3Gsi27uiR5xfZZvLcEaNk26S1RAT_i_oVGKf0oJ9SP0NDL_pLOv2rI2p9HhEBEjUXsr_RnEA5XMrQrpgo?key=MYfJe5ipT2YlIC4yxeebDQ)

## 5. Use a técnica correta

- [ ]  Escreva em linguagem impessoal e informativa
- [ ]  Use [seo-copywriting](https://docs.google.com/document/d/1cNc6w0H9Tq0gVW5nCYEEU2RSvBtuPZ5cOj-zW3bq6ak/edit?usp=sharing) para escrever
- [ ]  Aplique a [legibilidade-para-blog](https://docs.google.com/document/d/1QA_23aYN2SwuWbCw97gUvc66ToW9VwBfOhYxMsX7M2E/edit?usp=sharing)

## 6. Escreva

- [ ]  Posicione os elementos da sua pesquisa ao longo no corpo do texto dentro dos subtópicos e conecte as frases com [lista-palavras-transição](https://docs.google.com/document/d/17ejHDg9JWrDC6VwlC4o8SvFByJjZWPjWPP9Rwla0Qbg/edit?usp=sharing)
- [ ]  Veja onde é possível organizar a informação no formato de lista
- [ ]  Defina em quais subtópicos você vai usar exemplos e metáforas
- [ ]  Sugira artigos para backlinks dentro do texto
- [ ]  Escreva a introdução (resumo do conceito principal + estrutura do texto)
- [ ]  Escreva o texto do resumo com até 160 caracteres (meta descrição)
- [ ]  Defina a imagem de destaque de acordo com o branding da marca
- [ ]  Use a CTA padrão do nosso template para finalizar

## 7. Revise

- [ ]  Ortografia
- [ ]  Gramática
- [ ]  Digitação
- [ ]  Estrutura do texto
- [ ]  Palavras-chaves
- [ ]  Legibilidade
- [ ]  Formatação
- [ ]  Didática
- [ ]  Originalidade

## 8. Entregue o texto

- [ ]  Marque a tarefa como concluída no Notion
- [ ]  Siga para a próxima tarefa
