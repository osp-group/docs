O [Driva.IO](http://Driva.IO) é a ferramenta que usamos para criar listas com leads do outbound, para criar uma nova lista, usamos um filtro pré-definido, que se encaixa dentro do ICP que estamos buscando, selecionamos a loalização que iremos trabalhar e exportamos uma nova lista, através através da opção de Enviar ao CRM, que já está conectada ao Exact Sales. Interessante enviar a lista para validação no grupo Vendas/MKT.
