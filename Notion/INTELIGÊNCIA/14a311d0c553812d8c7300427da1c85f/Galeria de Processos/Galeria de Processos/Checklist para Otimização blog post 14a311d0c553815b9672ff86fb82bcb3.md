Status: Concluído

A última etapa do conteúdo é a publicação otimizada para SEO. Siga o passo a passo abaixo.

## **1. Otimização**

- [ ]  Identifique e configure h1, h2, h3, h4, parágrafos, listas, citações, imagens

[](https://lh7-us.googleusercontent.com/docsz/AD_4nXfcSnAHqYZs_GcfpWndTqSZIhGSHtWJggyWWaEilXaKvD9TsNPnmxD6409nl0grW1cGmTUah6dHIHi40aCaS-RX2I5yh0UchNZXVaIM0ry15qd8a7pVpm8ZLjpzSpPEFytUncoxVjkWzUuaqxtFwJFpNwaWfleqy_huWeBc8cH8zlCMmQ05Gw?key=n6GvVZHNY3lgU8chtJo8Eg)

- [ ]  Verificar se a keyword principal está presente na title, slug, meta e h1
- [ ]  Colocar a parte mais importante da primeira frase do texto em negrito
- [ ]  Dividir os parágrafos em no máximo 4 linhas
- [ ]  Centralizar imagens e vídeos, quando necessário
- [ ]  Verificar os slugs e tamanho das imagens

## **2. Link Building**

- [ ]  Fazer linkagem interna para outros conteúdos relacionados (mín. 3)
- [ ]  O texto âncora não pode ser repetido e nem genérico, pois deve ter conexão com o assunto
- [ ]  (Exemplo do que não pode: “Veja mais”, “Você também pode gostar" etc)
- [ ]  Configurar o link para 'abrir em uma nova guia':

[](https://lh7-us.googleusercontent.com/docsz/AD_4nXc2Oe_400zt2NGwitJVhB26S-mTG9tHxAGkd4WCOipESn1W7zp2fsNLu-iMkVaPD7BSJGZTvgmOJYXQ94Zlump4Sg7upjwD-m_PFNIQSE1pnb0eF8fy7NyPI8JpcorFisuw3UicimqAcUHaQF4BAtipdLLCjUfwKqmhlm2HIG_XMnxleW9luxo?key=n6GvVZHNY3lgU8chtJo8Eg)

## **3. Wordpress**

- [ ]  Definir categoria
- [ ]  Definir resumo (max. 4 linhas, adaptação da meta)
- [ ]  Definir imagem destacada (1200 x 675 I 512 x 335 I max. 200kb)
- [ ]  Definir custom scripts, se houver

## **4. Compartilhamento**

- [ ]  Configurar pré-visualização do Google
- [ ]  Configurar pré-visualização do Facebook
- [ ]  Configurar pré-visualização do Twitter

## **5.  Últimos ajustes**

- [ ]  Exibir prévia no desktop
- [ ]  Verificar se todos os itens anteriores foram atendidos
- [ ]  Observar a escaneabilidade, se o texto tem o tamanho certo (nem mais, nem menos)
- [ ]  Publicar o post

## **6. Ux design**

- [ ]  Após publicar, confira a versão desktop e mobile
- [ ]  Confira a Categoria definida
- [ ]  Confira se Imagem de destaque está proporcional, tem a ver com o tema e ficou bem
- [ ]  Faça alterações, quando necessário
